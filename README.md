# Real Car ADAS Monitor

Многопоточная система помощи водителю (ADAS), объединяющая обработку данных телеметрии автомобиля (OBD-II) и мониторинг состояния водителя (DMS) в реальном времени с использованием веб-камеры.

## Описание системы

Приложение работает в двух параллельных потоках для обеспечения высокой производительности:
1. **Поток OBD (10 Hz):** Считывает данные автомобиля (скорость, обороты, дроссельная заслонка и т.д.) из CSV-файла и классифицирует стиль вождения (SLOW, NORMAL, AGGRESSIVE) с помощью нейронной сети в формате ONNX.
2. **Основной поток (Main):** Захватывает видео с веб-камеры, выполняет поиск лица (OpenCV DNN) и анализ глаз (Haar Cascades). Алгоритм фиксирует отвлечение внимания (поворот головы) и усталость (закрытые глаза).

Оба потока обмениваются данными через потокобезопасную структуру `SharedState`. Итоговый результат выводится на комбинированную приборную панель, параллельно записывается видео сессии и ведется текстовый лог событий.

## Стек технологий

| Технология | Назначение |
| --- | --- |
| **C++ 17** | Основной язык разработки, многопоточность (`std::thread`, `std::mutex`, `std::atomic`) |
| **OpenCV 4.x** | Захват видео, отрисовка интерфейса, детекция лиц (DNN) и глаз (CascadeClassifier) |
| **ONNX Runtime** | Выполнение инференса модели машинного обучения для классификации |
| **GoogleTest** | Unit-тестирование модулей |
| **Doxygen** | Автоматическая генерация HTML-документации по исходному коду |
| **CMake** | Кроссплатформенная сборка проекта |

## Инструкция по сборке

1. Создайте директорию для сборки и перейдите в нее:
   ```bash
   mkdir build && cd build
   ```
2. Сгенерируйте конфигурационные файлы (укажите ваш путь к OpenCV):
   ```bash
   cmake .. -DOpenCV_DIR="C:/opencv/build"
   ```
3. Соберите проект в конфигурации Release:
   ```bash
   cmake --build . --config Release
   ```
*Примечание: Убедитесь, что библиотеки `opencv_world4xx.dll` и `onnxruntime.dll` находятся в папке `build/Release` рядом с исполняемым файлом.*

## Как запустить

Запуск системы осуществляется из директории `build`:
```bash
.\Release\RealCarMonitor.exe
```

**Управление в программе:**
* `Q` или `ESC` — Завершить работу программы и вывести финальную статистику в консоль.
* `SPACE` — Поставить систему на паузу / Возобновить работу.
* `S` — Сделать скриншот (сохраняется в папку `output/`).
