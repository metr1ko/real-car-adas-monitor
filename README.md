Real Car ADAS Monitor

Многопоточная система помощи водителю (ADAS), объединяющая обработку данных телеметрии автомобиля (OBD-II) и мониторинг состояния водителя (DMS) в реальном времени с использованием веб-камеры.

Описание системы

Приложение работает в двух параллельных потоках для обеспечения высокой производительности:

Поток OBD (10 Hz): Считывает данные автомобиля (скорость, обороты, дроссельная заслонка и т.д.) из CSV-файла и классифицирует стиль вождения (SLOW, NORMAL, AGGRESSIVE) с помощью нейронной сети в формате ONNX.

Основной поток (Main): Захватывает видео с веб-камеры, выполняет поиск лица (OpenCV DNN) и анализ глаз (Haar Cascades). Алгоритм фиксирует отвлечение внимания (поворот головы) и усталость (закрытые глаза).

Оба потока обмениваются данными через потокобезопасную структуру SharedState. Итоговый результат выводится на комбинированную приборную панель, параллельно записывается видео сессии и ведется текстовый лог событий.

Инструкция по сборке

Создайте директорию для сборки и перейдите в нее:

mkdir build && cd build


Сгенерируйте конфигурационные файлы (укажите ваш путь к OpenCV):

cmake .. -DOpenCV_DIR="C:/opencv/build"


Соберите проект в конфигурации Release:

cmake --build . --config Release


Примечание: Убедитесь, что библиотеки opencv_world4xx.dll и onnxruntime.dll находятся в папке build/Release рядом с исполняемым файлом.

Как запустить

Запуск системы осуществляется из директории build:

.\Release\RealCarMonitor.exe


Управление в программе:

Q или ESC — Завершить работу программы и вывести финальную статистику в консоль.

SPACE — Поставить систему на паузу / Возобновить работу.

S — Сделать скриншот (сохраняется в папку output/).
