Архитектура системы Real Car ADAS Monitor
Общее описание
Система представляет собой многопоточное C++ приложение для мониторинга состояния автомобиля и водителя в режиме реального времени. Приложение объединяет данные автомобильной телеметрии, нейросетевую классификацию стиля вождения и компьютерное зрение для отслеживания состояния водителя.

Основные модули
OBDParser — модуль чтения и парсинга CSV-файлов с телеметрией автомобиля.

ONNXClassifier — модуль загрузки ИИ-модели и классификации стиля вождения (SLOW/NORMAL/AGGRESSIVE).

Dashboard — графический модуль отрисовки приборной панели с помощью OpenCV.

DMSMonitor — модуль системы мониторинга водителя, анализирующий кадры с веб-камеры на наличие лица, открытость глаз и поворот головы.

DMSHUD — графический модуль наложения алертов (усталость, отвлечение) на кадр с камеры.
